# Minimal-UI

A minimalist, high-performance CSS framework designed for rapid prototyping, inspired by the **s17 Labs** aesthetic. 

Building efficient, lightweight software for the modern web requires a foundation that doesn't get in the way. `minimal-ui` provides the essential components and utility classes to go from an empty HTML file to a sleek, dark-mode interface in minutes.

---

## Quick Start

Drop this link into the `<head>` of your HTML file to instantly apply:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/s17labs/minimal-ui@main/minimal-ui.css">
```
## Requirements

FontAwesome:
```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/7.2.0/css/all.min.css">
```

Google Font - Aldrich:
```html
<link href="https://fonts.googleapis.com/css2?family=Aldrich&display=swap" rel="stylesheet">
```
